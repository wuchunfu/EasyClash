# EasyClash

An easy-to-use GUI for [clash](https://github.com/Dreamacro/clash) / [Clash.Meta](https://github.com/MetaCubeX/Clash.Meta).

Supported: 
- `Macos Intel(amd)/M1(arm)`, 
- `Windows arm/amd`
- `Linux (in process)`

[Donwload](https://github.com/daodao97/EasyClash/releases)
